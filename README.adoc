# Purpose

This image reads mqtt messages from a mqtt broker and inserts it into a timescaledb.

# Configuration (Enviroment variables)

```
ENV MQTT_HOST=mqtt
ENV MQTT_PORT=1883
ENV PGHOST=localhost
ENV PGUSER=postgres
ENV PGDATABASE=postgres
ENV PGPASSWORD=
```
